# Section 1 — Approved Tools

**Approved for production use:**
- **Claude for Work** (enterprise license, procured Q1 2025) — approved for all use cases described in Section 3.
- **Microsoft Copilot for M365** — approved for use within the Microsoft 365 environment (Outlook, Word, Excel) subject to the constraints in Section 2.

**Not approved for use with bank data:**
- Any personal-account AI service (ChatGPT, Claude.ai personal, Gemini, etc.) — regardless of subscription tier.
- Any AI tool that processes data outside of bank-approved infrastructure.
- Browser extensions that send page content to external AI services.

**Pending evaluation (not yet approved):**
- AI-native document review tools.
- Automated covenant monitoring systems.
- Any tool that integrates directly with core banking systems (Fiserv, nCino).
