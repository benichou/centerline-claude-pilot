# What I Wish I Had

- Something that automatically flags when I haven't contacted a borrower in X days.
- A way to search across all my emails for a specific borrower without manually scrolling through Outlook.
- A template library I can update and reuse instead of re-writing prompts from scratch every time.
- Something that reads the covenant package and fills in my tracking spreadsheet automatically.

I've tried building some of this but I don't have time to do it right. If the bank is going to roll out Claude I'd want to be involved in designing these workflows. The RM perspective is completely missing from how these systems usually get built.

*Last updated: March 28, 2025 — T. DeLuca, RM, Commercial Banking.*
